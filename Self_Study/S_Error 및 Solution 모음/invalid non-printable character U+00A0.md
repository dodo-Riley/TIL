# invalid non-printable character U+00A0

### 1. 상황

> `Notion`에서 입력한 코드 내용을 그대로 `ctrl+c`/`ctrl+v`해서 `Spyder`에서 실행
> 

### 2. 원인

> 복사한 코드에 존재하는 공백이 `Python`에서 공백과 형식이 다르기 때문에 오류 발생
> 

### 3. 해결책

> 복사한 코드 내의 공백을 지우고 다시 입력
>
