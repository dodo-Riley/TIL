## [3] vscode 설치 확인

- `Windows`
    1. vscode 실행을 원하는 폴더 내에서 마우스 우클릭을 통해 `Code(으)로 열기` 옵션이 있는 지 확인합니다. **(만약 해당 옵션이 없다면 vscode를 재 설치해야 합니다.)**
        
        ![Untitled](Visual%20Studio%20Code%20%E1%84%89%E1%85%A5%E1%86%AF%E1%84%8E%E1%85%B5%20107f03ef9cfb46e69f22a3574f6c694b/Untitled%202.png)
        
    2. `Code(으)로 열기` 옵션을 클릭 했을 때 vscode가 정상적으로 실행되는 지 확인합니다.
        
        ![빨간색 동그라미 부분에 vscode를 열었던 폴더 이름이 잘 나타나는 지 확인합니다.](Visual%20Studio%20Code%20%E1%84%89%E1%85%A5%E1%86%AF%E1%84%8E%E1%85%B5%20107f03ef9cfb46e69f22a3574f6c694b/Untitled%203.png)
        
        빨간색 동그라미 부분에 vscode를 열었던 폴더 이름이 잘 나타나는 지 확인합니다.
        
- `Mac`
    - vscode 를 성공적으로 설치 하더라도, 다음 그림과 같이 특정 폴더에서 우클릭(control + click)을 눌렀을 때, `Code(으)로 열기` 가 보이지 않을 수 있습니다.
        
        ![Untitled](Visual%20Studio%20Code%20%E1%84%89%E1%85%A5%E1%86%AF%E1%84%8E%E1%85%B5%20107f03ef9cfb46e69f22a3574f6c694b/Untitled%204.png)
        
    - 이 경우 아래와 같은 과정을 따라합니다.
    1. Spotlight (command + 스페이스 바) 에서 `Automator.app` 을 검색합니다.
        
        ![Untitled](Visual%20Studio%20Code%20%E1%84%89%E1%85%A5%E1%86%AF%E1%84%8E%E1%85%B5%20107f03ef9cfb46e69f22a3574f6c694b/Untitled%205.png)
        
    2. `빠른 동작`을 클릭하고 선택을 눌러줍니다.
        
        ![Untitled](Visual%20Studio%20Code%20%E1%84%89%E1%85%A5%E1%86%AF%E1%84%8E%E1%85%B5%20107f03ef9cfb46e69f22a3574f6c694b/Untitled%206.png)
        
    3. 이후 아래 화면과 같이 세팅해 줍니다.
        
        `위쪽 영역`
        
        - 현재 수신하는 작업 흐름 : 파일 또는 폴더
        - 선택 항목 위치 : Finder
        
        `쉘 스크립트 영역`
        
        - 쉘 스크립트 실행 더블클릭 후 오른쪽에 쉘 스크립트  실행 영역이 생긴 것을 확인합니다.
        - 쉘을 `/bin/bash` 로 설정해 주시고, 통과 입력을 `변수` 로 지정해 주세요.
        - `open -n -b "com.mcrosoft.VSCode" --args "$*”` 명령어를 그 아래 적습니다.
        
        ![Untitled](Visual%20Studio%20Code%20%E1%84%89%E1%85%A5%E1%86%AF%E1%84%8E%E1%85%B5%20107f03ef9cfb46e69f22a3574f6c694b/Untitled%207.png)
        
    4. Command + s 를 눌러 원하는 이름을 설정해 동작을 지정한 후 저장합니다.
        
        ![Untitled](Visual%20Studio%20Code%20%E1%84%89%E1%85%A5%E1%86%AF%E1%84%8E%E1%85%B5%20107f03ef9cfb46e69f22a3574f6c694b/Untitled%208.png)
        
1. 이제 특정 폴더나 파일을 Vscode 로 열 수 있습니다.
